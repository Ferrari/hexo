建立

安裝完成後，在你喜愛的資料夾下，執行以下指令，Hexo 即會自動在目標資料夾建立網站所需要的所有檔案。 

hexo init <folder>

建立完成後，資料夾結構如下： 

|-- .gitignore
|-- _config.yml
|-- package.json
|-- source
|   |-- _posts
        |-- hello-world.md
|   |-- _drafts
|-- themes
    |-- light

_config.yml

全域的設定檔案。

package.json

應用程式資料。請勿刪除，若不幸刪除，請依照以下內容重建檔案。

source

在此資料夾內的檔案會被處理並另存至public資料夾。名稱開頭為.或_的檔案或資料夾會被忽略，除了_posts資料夾以外。

themes

主題資料夾。Hexo 的預設主題為 Light。
